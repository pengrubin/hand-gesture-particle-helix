# 跨平台手势识别系统依赖
# 支持 Python 3.8-3.12 和 macOS/Windows

# 核心依赖 - 使用兼容版本范围
opencv-python>=4.5.0,<5.0.0
mediapipe>=0.9.0,<=0.10.21
numpy>=1.21.0,<2.0.0

# 音频处理（可选）
pygame>=2.1.0,<3.0.0

# 3D渲染（可选）
PyOpenGL>=3.1.5,<4.0.0
# PyOpenGL-accelerate>=3.1.5,<4.0.0  # Intel Mac 可能不支持，注释掉避免安装失败

# MIDI处理（可选）
mido>=1.2.0,<2.0.0

# 平台特定的额外依赖说明：
# 
# Windows:
# - 可能需要安装 Microsoft Visual C++ Redistributable
# - 某些情况下需要: pip install opencv-python-headless 替代 opencv-python
#
# macOS Intel:
# - 如果遇到 PyOpenGL-accelerate 安装问题，跳过此包
# - 使用: pip install --no-deps PyOpenGL-accelerate 或完全跳过
#
# macOS Apple Silicon:
# - 推荐使用 conda 环境: conda install opencv mediapipe numpy
# - 或使用预编译轮子: pip install --only-binary=all -r requirements_cross_platform.txt
#
# 安装命令:
# pip install -r requirements_cross_platform.txt
#
# 如果遇到编译错误，尝试只安装二进制包：
# pip install --only-binary=all opencv-python mediapipe numpy pygame PyOpenGL mido
